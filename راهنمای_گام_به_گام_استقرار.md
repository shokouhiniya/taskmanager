## مرحله 1️⃣: تنظیم DNS Domain

### 1.1 ورود به پنل Domain
به پنل مدیریت domain خود (مثلاً `mardomi.org`) وارد شوید.

### 1.2 اضافه کردن A Record
یک رکورد جدید اضافه کنید:

```
Type: A
Name: taskmanager
Value: [IP سرور شما]
TTL: 3600
```

مثال:
```
Type: A
Name: taskmanager
Value: 185.123.45.67
TTL: 3600
```

### 1.3 تست DNS
بعد از 5-10 دقیقه، تست کنید:

```bash
ping taskmanager.mardomi.org
```

باید IP سرور شما رو نشون بده.

## مرحله 7️⃣: تنظیم Nginx

### 7.1 ایجاد فایل تنظیمات Nginx

```bash
sudo nano /etc/nginx/sites-available/taskmanager
```

محتویات زیر را وارد کنید:

```nginx
server {
    listen 80;
    server_name taskmanager.mardomi.org;

    # Frontend - صفحه اصلی
    location / {
        proxy_pass http://localhost:4173;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }

    # Backend API
    location /api/ {
        proxy_pass http://localhost:3000/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
        
        # CORS headers
        add_header 'Access-Control-Allow-Origin' '*' always;
        add_header 'Access-Control-Allow-Methods' 'GET, POST, PUT, DELETE, OPTIONS' always;
        add_header 'Access-Control-Allow-Headers' 'Authorization, Content-Type' always;
    }
}
```

ذخیره کنید: `Ctrl+X` → `Y` → `Enter`

### 7.2 فعال‌سازی سایت

```bash
# ایجاد symlink
sudo ln -s /etc/nginx/sites-available/taskmanager /etc/nginx/sites-enabled/

# حذف default site (اختیاری)
sudo rm /etc/nginx/sites-enabled/default
```

### 7.3 تست تنظیمات Nginx

```bash
sudo nginx -t
```

باید پیام `syntax is ok` و `test is successful` رو ببینید.

### 7.4 Restart کردن Nginx

```bash
sudo systemctl restart nginx
```

### 7.5 بررسی وضعیت Nginx

```bash
sudo systemctl status nginx
```

## مرحله 8️⃣: تنظیم Firewall

### 8.1 باز کردن پورت‌های لازم

```bash
# اگر UFW نصب نیست
sudo apt install -y ufw

# باز کردن پورت‌ها
sudo ufw allow 22/tcp    # SSH
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS (برای آینده)

# فعال‌سازی Firewall
sudo ufw enable

# بررسی وضعیت
sudo ufw status
```
