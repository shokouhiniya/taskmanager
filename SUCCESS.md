### Clone و Setup
```bash
# Clone
git clone https://github.com/shokouhiniya/taskmanager.git
cd taskmanager

# Backend
cd backend
npm install
cp .env.example .env  # باید .env.example بسازی
npm run start:dev

# Frontend
cd ../frontend
npm install
npm run dev

# ngrok
ngrok http 5173
```
