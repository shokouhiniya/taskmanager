# 👥 راهنمای مدیریت کاربران

## ✅ قابلیت‌های جدید

### 1. ایجاد کاربر با بررسی تکراری
- چک می‌کنه username تکراری نباشه
- چک می‌کنه phone تکراری نباشه
- پیام خطای واضح نشون میده

### 2. ویرایش کاربر
- دکمه ✏️ ویرایش برای هر کاربر
- می‌تونی phone، name، role رو تغییر بدی
- می‌تونی password رو عوض کنی (اختیاری)
- username قابل تغییر نیست

### 3. تغییر سریع نقش
- dropdown مستقیم روی کارت کاربر
- بدون نیاز به ویرایش کامل
- تغییر فوری نقش کاربر

## 🎯 نحوه استفاده

### ایجاد کاربر جدید
1. روی دکمه "➕ جدید" کلیک کن
2. فرم رو پر کن:
   - نام کاربری (یونیک)
   - رمز عبور
   - شماره موبایل (یونیک)
   - نام و نام خانوادگی
   - نقش کاربری
3. روی "✅ ایجاد کاربر" کلیک کن

### ویرایش کاربر
1. روی دکمه "✏️ ویرایش" کلیک کن
2. اطلاعات رو تغییر بده:
   - شماره موبایل
   - نام و نام خانوادگی
   - نقش کاربری
   - رمز عبور (اختیاری - خالی بذار اگر نمی‌خوای عوض کنی)
3. روی "✅ ذخیره تغییرات" کلیک کن

### تغییر سریع نقش
1. dropdown نقش رو باز کن
2. نقش جدید رو انتخاب کن
3. خودکار ذخیره میشه

### حذف کاربر
1. روی دکمه "🗑️ حذف" کلیک کن
2. تایید کن
3. کاربر حذف میشه

## ⚠️ خطاهای رایج

### "این نام کاربری قبلاً استفاده شده است"
**راه حل:** یه نام کاربری دیگه انتخاب کن

### "این شماره موبایل قبلاً ثبت شده است"
**راه حل:** یه شماره دیگه وارد کن یا کاربر قبلی رو ویرایش کن

### "UNIQUE constraint failed"
**راه حل:** این خطا دیگه نباید بیاد - کد آپدیت شده

## 🔐 نکات امنیتی

1. **Password ها hash میشن** - با bcrypt
2. **Username یونیک است** - نمیشه تکراری باشه
3. **Phone یونیک است** - نمیشه تکراری باشه
4. **Password اختیاری در ویرایش** - اگر خالی بذاری، عوض نمیشه

## 📱 بهینه‌سازی موبایل

- فرم در یک ستون
- دکمه‌ها تمام عرض
- dropdown نقش روی کارت
- دکمه‌های ویرایش و حذف کنار هم
- فونت‌های کوچک‌تر برای موبایل

همه چیز آماده است! 🚀
