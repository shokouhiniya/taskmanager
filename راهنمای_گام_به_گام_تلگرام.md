# 📱 راهنمای گام‌به‌گام راه‌اندازی Telegram Mini App

## قبل از شروع

این راهنما برای کسانی است که می‌خواهند سیستم را به صورت Telegram Mini App اجرا کنند.

**نیازمندی‌ها:**
- ✅ Node.js نصب شده
- ✅ npm نصب شده
- ✅ ربات تلگرام (از @BotFather)
- ✅ ngrok نصب شده

---

## گام 1️⃣: نصب ngrok

### ویندوز:

**روش 1 - با Chocolatey:**
```bash
choco install ngrok
```

**روش 2 - دانلود مستقیم:**
1. برو به https://ngrok.com/download
2. فایل ZIP را دانلود کن
3. Extract کن
4. `ngrok.exe` را به پوشه `C:\Windows\System32` کپی کن

**ثبت‌نام و دریافت Token:**
1. برو به https://dashboard.ngrok.com/signup
2. ثبت‌نام کن (رایگان)
3. از صفحه Dashboard، authtoken را کپی کن
4. در ترمینال:
   ```bash
   ngrok config add-authtoken YOUR_TOKEN_HERE
   ```

## گام 4️⃣: راه‌اندازی ngrok

**ترمینال 3 (جدید):**
```bash
ngrok http 5173
```

**انتظار:**
```
Forwarding: https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:5173
```

**مهم:** این URL را کپی کنید! مثلاً:
```
https://a1b2-12-34-56-78.ngrok-free.app
```

## گام 5️⃣: تنظیم Bot در BotFather

### 5.1 باز کردن BotFather
1. در تلگرام به [@BotFather](https://t.me/BotFather) بروید
2. دستور `/mybots` را بزنید

### 5.2 انتخاب Bot
- اگر ربات دارید: روی نام ربات کلیک کنید
- اگر ندارید: دستور `/newbot` بزنید و یک ربات بسازید

### 5.3 تنظیم Menu Button
1. روی "Bot Settings" کلیک کنید
2. روی "Menu Button" کلیک کنید
3. روی "Configure Menu Button" کلیک کنید
4. **URL ngrok را وارد کنید:**
   ```
   https://a1b2-12-34-56-78.ngrok-free.app
   ```
5. متن دکمه را وارد کنید:
   ```
   🚀 ورود به سامانه
   ```
6. "Done" را بزنید

## گام 6️⃣: آپدیت Backend

### 6.1 ویرایش .env
فایل `backend/.env` را باز کنید و URL ngrok را اضافه کنید:

```env
JWT_SECRET=your-secret-key
JWT_EXPIRES_IN=7d
PORT=3000
TELEGRAM_BOT_TOKEN=8787350194:AAHzqvnJhvPOh1Fg6I7L8dEbav95N6e1u04
WEBAPP_URL=https://a1b2-12-34-56-78.ngrok-free.app
```

**مهم:** URL خودتان را جایگزین کنید!

### 6.2 Restart Backend
در ترمینال Backend:
1. `Ctrl+C` برای توقف
2. `npm run start:dev` برای شروع مجدد

## 🐛 مشکلات رایج و راه‌حل

### مشکل 1: Mini App باز نمی‌شود

**علائم:**
- بعد از کلیک روی دکمه، هیچ اتفاقی نمی‌افتد
- یا صفحه سفید نمایش داده می‌شود

**راه‌حل:**
```bash
# 1. بررسی ngrok
# آیا در حال اجرا است؟
# URL را دوباره چک کنید

# 2. بررسی URL در BotFather
# باید HTTPS باشد
# بدون / در انتها

# 3. تست URL در مرورگر
# URL ngrok را در مرورگر باز کنید
# باید صفحه لاگین نمایش داده شود
```

### مشکل 2: خطای "ngrok not found"

**راه‌حل:**
```bash
# نصب ngrok
choco install ngrok

# یا دانلود از:
https://ngrok.com/download

# بعد از نصب:
ngrok config add-authtoken YOUR_TOKEN
```

### مشکل 3: خطای احراز هویت

**علائم:**
- صفحه "خطا در احراز هویت" نمایش داده می‌شود

**راه‌حل:**
```bash
# 1. بررسی Token در .env
cat backend/.env | grep TELEGRAM_BOT_TOKEN

# 2. Backend را restart کنید
cd backend
# Ctrl+C
npm run start:dev

# 3. Cache مرورگر را پاک کنید
# در Mini App: بستن و باز کردن مجدد
```

### مشکل 4: Bottom Navigation نمایش داده نمی‌شود

**راه‌حل:**
```bash
# 1. مطمئن شوید از Telegram باز کرده‌اید
# 2. Frontend را restart کنید
cd frontend
# Ctrl+C
npm run dev

# 3. Mini App را ببندید و دوباره باز کنید
```

### مشکل 5: ngrok URL تغییر می‌کند

**علت:**
ngrok رایگان هر بار URL جدید می‌دهد.

**راه‌حل:**
```bash
# هر بار که ngrok را restart می‌کنید:
# 1. URL جدید را کپی کنید
# 2. در BotFather آپدیت کنید
# 3. در backend/.env آپدیت کنید
# 4. Backend را restart کنید

# یا:
# ngrok Pro بخرید (URL ثابت)
```

## 🎓 نکات مهم

### 1. ngrok باید همیشه در حال اجرا باشد
- اگر ngrok را ببندید، Mini App کار نمی‌کند
- URL جدید باید در BotFather آپدیت شود

### 2. Backend باید restart شود
- بعد از تغییر .env
- بعد از تغییر کد Telegram

### 3. Cache مرورگر
- اگر تغییری ندیدید، Cache را پاک کنید
- یا Mini App را ببندید و دوباره باز کنید

### 4. دو حالت کار می‌کند
- **Web:** با مرورگر عادی (`http://localhost:5173`)
- **Telegram:** با Mini App در تلگرام
