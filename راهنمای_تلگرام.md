# 🤖 راهنمای کامل Telegram Mini App

## 📱 Telegram Mini App چیست؟

Telegram Mini App یک وب‌اپلیکیشن است که داخل تلگرام باز می‌شود و به کاربران اجازه می‌دهد بدون خروج از تلگرام، از سیستم استفاده کنند.

### مزایا:
- ✅ احراز هویت خودکار با Telegram
- ✅ نیازی به ثبت‌نام یا لاگین نیست
- ✅ دسترسی سریع از داخل تلگرام
- ✅ اعلان‌های فوری
- ✅ تجربه کاربری بهتر روی موبایل

---

## 🚀 راه‌اندازی سریع

### مرحله 1: نصب ngrok

ngrok یک تونل HTTPS برای localhost شما ایجاد می‌کند.

**ویندوز:**
```bash
# با Chocolatey:
choco install ngrok

# یا دانلود مستقیم:
https://ngrok.com/download
```

**بعد از نصب:**
```bash
# ثبت‌نام در ngrok.com و دریافت authtoken
ngrok config add-authtoken YOUR_AUTHTOKEN
```

### مرحله 2: راه‌اندازی سرورها

**روش آسان (با اسکریپت):**
```bash
# ویندوز:
start-telegram-app.bat

# لینوکس/مک:
./start-telegram-app.sh
```

**روش دستی:**

**ترمینال 1 - Backend:**
```bash
cd backend
npm run start:dev
```

**ترمینال 2 - Frontend:**
```bash
cd frontend
npm run dev
```

**ترمینال 3 - ngrok:**
```bash
ngrok http 5173
```

### مرحله 3: کپی کردن URL ngrok

از پنجره ngrok، URL را کپی کنید:
```
Forwarding: https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:5173
```

URL مورد نیاز: `https://xxxx-xx-xx-xx-xx.ngrok-free.app`

### مرحله 4: تنظیم Bot در BotFather

1. به [@BotFather](https://t.me/BotFather) بروید
2. دستور `/mybots` را بزنید
3. ربات خود را انتخاب کنید (یا با `/newbot` یکی بسازید)
4. "Bot Settings" → "Menu Button" → "Configure Menu Button"
5. URL ngrok را وارد کنید
6. متن دکمه: `🚀 ورود به سامانه`

### مرحله 5: آپدیت Backend

فایل `backend/.env` را باز کنید:

```env
JWT_SECRET=your-secret-key
JWT_EXPIRES_IN=7d
PORT=3000
TELEGRAM_BOT_TOKEN=8787350194:AAHzqvnJhvPOh1Fg6I7L8dEbav95N6e1u04
WEBAPP_URL=https://xxxx-xx-xx-xx-xx.ngrok-free.app
```

**مهم:** Backend را restart کنید (Ctrl+C و دوباره `npm run start:dev`)

### مرحله 6: تست

1. به ربات خود در تلگرام بروید
2. دستور `/start` را بزنید
3. روی دکمه "🚀 ورود به سامانه" کلیک کنید
4. Mini App باز می‌شود و شما لاگین می‌شوید!

## 🎯 نحوه استفاده

### برای کاربران عادی (Reporter):

1. **ورود:**
   - دستور `/start` در ربات
   - کلیک روی "🚀 ورود به سامانه"
   - لاگین خودکار

2. **ثبت خبر:**
   - از Bottom Navigation روی "📝 ثبت خبر" کلیک کنید
   - فرم را انتخاب کنید
   - فیلدها را پر کنید
   - ثبت کنید

3. **مشاهده خبرها:**
   - از Bottom Navigation روی "📊 خبرها" کلیک کنید
   - خبرهای خود را ببینید

4. **خبرهای ارجاع شده:**
   - از Bottom Navigation روی "📬 ارجاعی" کلیک کنید
   - خبرهای محول شده را ببینید

5. **اقدامات:**
   - از Bottom Navigation روی "⚡ اقدامات" کلیک کنید
   - اقدامات محول شده را مدیریت کنید

### برای ادمین:

همه قابلیت‌های Reporter + :

- **مدیریت فرم‌ها:** Bottom Nav → 🎨 فرم‌ها
- **مدیریت کاربران:** Bottom Nav → 👥 کاربران
- **تایید/رد خبرها:** در لیست خبرها
- **ارجاع خبر:** دکمه "📬 تایید و ارجاع"

## 🔐 امنیت

### احراز هویت Telegram

سیستم از روش رسمی Telegram برای احراز هویت استفاده می‌کند:

1. Telegram initData را دریافت می‌کند
2. با HMAC-SHA256 اعتبارسنجی می‌شود
3. اگر معتبر باشد، کاربر لاگین می‌شود

### نقش‌های کاربری

- **Reporter (پیش‌فرض):** کاربران جدید Telegram به صورت خودکار Reporter می‌شوند
- **Admin:** باید از پنل مدیریت کاربران، نقش را تغییر دهید

## 🎓 نکات مهم

1. **ngrok URL موقتی است:**
   - هر بار که ngrok را restart کنید، URL تغییر می‌کند
   - باید در BotFather آپدیت کنید

2. **HTTPS الزامی است:**
   - Telegram فقط HTTPS را قبول می‌کند
   - برای localhost از ngrok استفاده کنید

3. **احراز هویت دوگانه:**
   - سیستم هم از username/password پشتیبانی می‌کند
   - هم از Telegram authentication

4. **نقش پیش‌فرض:**
   - کاربران جدید Telegram به صورت خودکار Reporter می‌شوند
   - ادمین باید نقش را تغییر دهد

## 🆘 کمک و پشتیبانی

### مشکلات رایج:

**Mini App باز نمی‌شود:**
- بررسی کنید ngrok در حال اجرا است
- URL در BotFather درست تنظیم شده
- HTTPS دارد (نه HTTP)

**خطای احراز هویت:**
- Token در .env درست است
- Backend restart شده
- Console را بررسی کنید (F12)

**Bottom Navigation نمایش داده نمی‌شود:**
- مطمئن شوید در Telegram باز کرده‌اید
- Cache مرورگر را پاک کنید
