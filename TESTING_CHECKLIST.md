## 🐛 گزارش باگ

اگر مشکلی پیدا کردید، اطلاعات زیر را جمع‌آوری کنید:

1. **مراحل بازتولید:**
   - چه کاری انجام دادید؟
   - چه اتفاقی افتاد؟
   - چه انتظاری داشتید؟

2. **Console Errors:**
   - F12 → Console
   - Screenshot یا کپی خطاها

3. **Network Errors:**
   - F12 → Network
   - بررسی درخواست‌های ناموفق (قرمز)
   - کپی کردن Request/Response

4. **Backend Logs:**
   - خطاهای ترمینال backend
   - Stack trace اگر وجود دارد
