# 📱 راهنمای کامل Mini App برای بله

## 🎯 خلاصه

این سیستم حالا از **هر دو پلتفرم Telegram و Bale** پشتیبانی می‌کند!

## ✨ ویژگی‌های جدید

- ✅ پشتیبانی کامل از بله
- ✅ تشخیص خودکار پلتفرم (Telegram یا Bale)
- ✅ احراز هویت جداگانه برای هر پلتفرم
- ✅ ذخیره کاربران با `baleId` یا `telegramId`
- ✅ استفاده همزمان از هر دو پلتفرم

## 🔧 تنظیمات اولیه

### 1. Backend

فایل `backend/.env` رو باز کن و این تنظیمات رو اضافه کن:

```env
# تنظیمات بله
BALE_BOT_TOKEN=407715407:wf0tQMl7yRbnqsOhkUUyIMxb7p8simgjGmg
BALE_API_URL=https://tapi.bale.ai

# آدرس Mini App (از ngrok)
WEBAPP_URL=https://your-ngrok-url.ngrok-free.dev
```

### 2. اجرای سرورها

```bash
# Backend
cd backend
npm run start:dev

# Frontend
cd frontend
npm run dev

# ngrok
ngrok http 5173
```

## 🤖 تنظیم ربات بله

### مرحله 1: ساخت ربات (اگر نساختی)

1. به [@BotFather](https://ble.ir/BotFather) در بله برو
2. دستور `/newbot` رو بفرست
3. نام ربات رو وارد کن (مثلاً: سامانه مدیریت)
4. username ربات رو وارد کن (مثلاً: taskmanager_bot)
5. توکن ربات رو دریافت کن

**توکن فعلی:** `407715407:wf0tQMl7yRbnqsOhkUUyIMxb7p8simgjGmg`

### مرحله 2: تنظیم Mini App

1. به ربات خودت در بله برو
2. دستور `/setminiapp` رو بفرست
3. URL ngrok رو وارد کن:
   ```
   https://your-ngrok-url.ngrok-free.dev
   ```
4. عنوان دکمه رو وارد کن:
   ```
   🚀 ورود به سامانه
   ```

### مرحله 3: تست

1. ربات رو در بله باز کن
2. `/start` رو بفرست
3. روی دکمه "🚀 ورود به سامانه" کلیک کن
4. Mini App باز میشه و خودکار وارد میشی!

## 🎨 تفاوت‌ها با Telegram

### Backend

| مورد | Telegram | Bale |

| متغیر محیطی | TELEGRAM_BOT_TOKEN | BALE_BOT_TOKEN |
| فیلد کاربر | telegramId | baleId |
| Endpoint | /auth/telegram | /auth/bale |

### Frontend

- تشخیص خودکار پلتفرم
- استفاده از endpoint مناسب
- UI یکسان برای هر دو

## 📱 نحوه کار

### 1. تشخیص پلتفرم

Frontend خودکار تشخیص میده که کاربر از Telegram اومده یا Bale:

```typescript
const isBale = window.location.hostname.includes('bale') || 
               WebApp.platform === 'bale';
```

### 2. احراز هویت

بسته به پلتفرم، endpoint مناسب صدا زده میشه:

- **Telegram:** `POST /auth/telegram`
- **Bale:** `POST /auth/bale`

### 3. ذخیره کاربر

کاربران با ID مخصوص پلتفرمشون ذخیره میشن:

- کاربران Telegram: `telegramId`
- کاربران Bale: `baleId`

## 🐛 رفع مشکلات

### خطا: "اطلاعات کاربر یافت نشد"

**علت:** از طریق دکمه Mini App ربات وارد نشدی

**راه حل:**
1. ربات رو در بله باز کن
2. `/start` رو بفرست
3. روی دکمه Mini App کلیک کن
4. URL رو مستقیم باز نکن!

### خطا: "Network Error"

**علت:** Backend یا Frontend در حال اجرا نیست

**راه حل:**
1. Backend رو چک کن: `http://localhost:3000`
2. Frontend رو چک کن: `http://localhost:5173`
3. ngrok رو restart کن

### خطا: "Bot Token Invalid"

**علت:** توکن ربات اشتباه است

**راه حل:**
1. فایل `.env` رو باز کن
2. `BALE_BOT_TOKEN` رو چک کن
3. Backend رو restart کن

### Mini App باز نمیشه

**راه حل:**
1. مطمئن شو ngrok در حال اجرا است
2. URL ngrok رو در ربات چک کن
3. URL باید با `https://` شروع شه
4. Backend و Frontend باید در حال اجرا باشن
