# 🔧 راهنمای رفع مشکل لاگین تلگرام

## ✅ تغییرات اعمال شده

### 1. رفع مشکل Password
- کاربران جدید تلگرام حالا با یک password تصادفی ایجاد می‌شوند
- دیگر خطای null password نمی‌دهد

### 2. رفع مشکل API Connection
- axios حالا در Telegram از `/api` proxy استفاده می‌کند
- vite proxy تمام درخواست‌های `/api/*` را به `localhost:3000` هدایت می‌کند
- دیگر نیازی به ngrok برای backend نیست

### 3. بهبود Error Handling
- console.log های دقیق برای debugging
- نمایش پیام خطا با WebApp.showAlert
- بررسی دقیق‌تر اطلاعات کاربر تلگرام

## 🚀 نحوه تست

### مرحله 1: اطمینان از اجرای سرورها
```bash
# Backend باید روی localhost:3000 در حال اجرا باشد
# Frontend باید روی localhost:5173 در حال اجرا باشد
```

### مرحله 2: اجرای ngrok
```bash
ngrok http 5173
```

### مرحله 3: تنظیم Bot
1. به [@BotFather](https://t.me/BotFather) بروید
2. `/mybots` → انتخاب ربات شما
3. Bot Settings → Menu Button → Configure Menu Button
4. URL: `https://your-ngrok-url.ngrok-free.dev`
5. Text: `🚀 ورود به سامانه`

### مرحله 4: تست
1. ربات را در تلگرام باز کنید
2. `/start` را بزنید
3. روی دکمه منو کلیک کنید
4. Mini App باز می‌شود

## 🔍 Debugging

اگر باز هم مشکل داشتید:

### 1. چک کردن Console
در تلگرام Desktop:
- Right Click → Inspect Element
- Console را باز کنید
- لاگ‌های زیر را ببینید:
  - `🔍 Telegram WebApp detected`
  - `📱 Platform: ...`
  - `👤 Telegram User: ...`
  - `🚀 Attempting Telegram login...`
  - `✅ Login successful: ...`

### 2. چک کردن Network
- Network tab را باز کنید
- درخواست به `/api/auth/telegram` را ببینید
- Status Code باید 200 یا 201 باشد

### 3. مشکلات رایج

#### خطا: "اطلاعات کاربر تلگرام یافت نشد"
**علت:** Mini App از طریق ربات باز نشده
**راه حل:** حتماً از طریق Menu Button ربات وارد شوید

#### خطا: "Network Error"
**علت:** Backend در دسترس نیست
**راه حل:** 
- Backend را restart کنید
- Frontend را restart کنید
- ngrok را restart کنید

#### خطا: "401 Unauthorized"
**علت:** مشکل در JWT token
**راه حل:**
- localStorage را پاک کنید
- دوباره وارد شوید

## 📋 چک لیست

- [ ] Backend روی localhost:3000 در حال اجرا است
- [ ] Frontend روی localhost:5173 در حال اجرا است
- [ ] ngrok در حال اجرا است و URL دارید
- [ ] URL در BotFather تنظیم شده
- [ ] از طریق Menu Button ربات وارد می‌شوید (نه مستقیم URL)
- [ ] Console را برای لاگ‌ها چک کرده‌اید

## 🎯 نکات مهم

1. **همیشه از طریق ربات وارد شوید** - نه با باز کردن مستقیم URL
2. **ngrok فقط برای frontend** - backend از طریق vite proxy در دسترس است
3. **Console را چک کنید** - تمام اطلاعات debugging در console است
4. **localStorage را پاک کنید** - اگر مشکل داشتید

## 🔄 Restart همه چیز

اگر همه چیز خراب شد:

```bash
# 1. Stop همه چیز
Ctrl+C در تمام ترمینال‌ها

# 2. پاک کردن localStorage
در Console تلگرام: localStorage.clear()

# 3. Start دوباره
# ترمینال 1
cd backend && npm run start:dev

# ترمینال 2  
cd frontend && npm run dev

# ترمینال 3
ngrok http 5173

# 4. تست دوباره
از طریق ربات وارد شوید
```
