# ⚡ شروع سریع Telegram Mini App

## 🎯 در 5 دقیقه راه‌اندازی کنید!

### پیش‌نیاز: نصب ngrok

**ویندوز:**
```bash
choco install ngrok
```

یا دانلود از: https://ngrok.com/download

**ثبت‌نام و تنظیم:**
```bash
# برو به https://dashboard.ngrok.com/signup
# بعد از ثبت‌نام، authtoken را کپی کن
ngrok config add-authtoken YOUR_TOKEN_HERE
```

---

## 🚀 راه‌اندازی (3 ترمینال)

### ترمینال 1️⃣ - Backend
```bash
cd backend
npm run start:dev
```
✅ منتظر بمانید تا ببینید: `🚀 Backend running on http://localhost:3000`

### ترمینال 2️⃣ - Frontend
```bash
cd frontend
npm run dev
```
✅ منتظر بمانید تا ببینید: `Local: http://localhost:5173`

### ترمینال 3️⃣ - ngrok
```bash
ngrok http 5173
```
✅ URL را کپی کنید: `https://xxxx-xx-xx-xx-xx.ngrok-free.app`

---

## 🤖 تنظیم Bot (2 دقیقه)

### 1. باز کردن BotFather
در تلگرام به [@BotFather](https://t.me/BotFather) بروید

### 2. تنظیم Menu Button
```
/mybots
→ انتخاب ربات
→ Bot Settings
→ Menu Button
→ Configure Menu Button
→ وارد کردن URL ngrok
→ متن: 🚀 ورود به سامانه
```

### 3. آپدیت Backend
فایل `backend/.env`:
```env
WEBAPP_URL=https://xxxx-xx-xx-xx-xx.ngrok-free.app
```

**Restart Backend:**
```bash
# در ترمینال Backend: Ctrl+C
npm run start:dev
```

## 🐛 مشکل دارید؟

### Mini App باز نمی‌شود:
```bash
# 1. بررسی ngrok
# آیا در حال اجرا است؟

# 2. تست URL در مرورگر
# URL ngrok را در Chrome باز کنید
# باید صفحه لاگین ببینید

# 3. بررسی BotFather
# URL درست تنظیم شده؟
# HTTPS دارد؟
```

### خطای احراز هویت:
```bash
# 1. Backend را restart کنید
cd backend
# Ctrl+C
npm run start:dev

# 2. Mini App را ببندید و دوباره باز کنید
```
