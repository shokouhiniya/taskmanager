## 🚀 راه‌اندازی سریع

### مرحله 1: نصب ngrok
```bash
choco install ngrok
ngrok config add-authtoken YOUR_TOKEN
```

### مرحله 2: راه‌اندازی سرورها
```bash
# ترمینال 1
cd backend
npm run start:dev

# ترمینال 2
cd frontend
npm run dev

# ترمینال 3
ngrok http 5173
```

### مرحله 3: کپی URL ngrok
```
https://xxxx-xx-xx-xx-xx.ngrok-free.app
```

### مرحله 4: تنظیم BotFather
1. [@BotFather](https://t.me/BotFather) → `/mybots`
2. انتخاب ربات
3. Bot Settings → Menu Button → Configure
4. وارد کردن URL ngrok
5. متن دکمه: `🚀 ورود به سامانه`

### مرحله 5: آپدیت Backend
فایل `backend/.env`:
```env
WEBAPP_URL=https://xxxx-xx-xx-xx-xx.ngrok-free.app
```
سپس Backend را restart کنید.

### مرحله 6: تست
1. به ربات در تلگرام بروید
2. دستور `/start`
3. کلیک روی دکمه
4. Mini App باز می‌شود!

## 🎓 نکات مهم

### 1. دو حالت کار می‌کند
- **Web:** `http://localhost:5173` (با username/password)
- **Telegram:** Mini App (با Telegram ID)

### 2. ngrok URL موقتی است
- هر بار restart، URL جدید
- باید در BotFather آپدیت شود

### 3. HTTPS الزامی
- Telegram فقط HTTPS قبول می‌کند
- برای localhost از ngrok استفاده کنید

### 4. نقش پیش‌فرض
- کاربران جدید Telegram: Reporter
- تغییر نقش: از پنل مدیریت
